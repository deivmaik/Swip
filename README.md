# Swip Landing Page

Swip lets you share all your social media with just a tap on the phone.

## Demo

This website is currently under build, for demo purposes please follow this link:
https://swip.netlify.app/

## Run Locally

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Tech Stack

**Client:**
React.js
,React Router
,React Icons
,Styled Components.
